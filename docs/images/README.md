# Documentation Images

This directory contains screenshots and visual assets for the GitHub README.

## Image Sources

All images are generated or captured from the live application:

- **banner.png**: Hero banner with branding
- **landing-page.png**: Main landing page view
- **streaming-summary.png**: Real-time streaming in action
- **ai-insights.png**: AI insights panel
- **highlights.png**: Smart highlights with provenance
- **demo-mode.gif**: Animated demo of streaming

## Image Guidelines

- All screenshots use dark mode for consistency
- Images are optimized for web (PNG for static, GIF for animated)
- Maximum width: 1200px for clarity on GitHub
- Maintain aspect ratios for professional appearance

## Regenerating Screenshots

To update screenshots:

1. Run the app: `npm run dev`
2. Navigate to relevant pages
3. Capture screenshots at 1200px width
4. Optimize with tools like TinyPNG
5. Replace files in this directory
